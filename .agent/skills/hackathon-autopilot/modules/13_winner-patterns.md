# Winner Patterns Mode

Goal: infer what "typically wins" and bias outputs toward those patterns.

Winning project patterns (use as heuristics, not external facts):

1. 10-second clarity: the judge instantly gets it
2. Strong narrative arc: problem → stakes → solution → impact
3. Demo reliability: offline-safe, precomputed fallback
4. Technical depth without risk: deterministic core + optional AI layer
5. Measurable impact metric: 1 killer metric + how measured
6. Non-obvious differentiation: not a generic chatbot
7. UX polish: a simple but premium user journey
8. Credible deployment story: who uses it tomorrow
9. "Why now": trend hook + urgency
10. Strong team execution plan: division of labor & milestones

Deliver:

- a "Winner Playbook" checklist
- how our chosen idea will hit each pattern
- a "Pattern Bonus" scoring rubric (0–2 each) for ideas:
  - Clarity, Differentiation, Demo Reliability, Depth-Control, Impact Metric, Deployability
