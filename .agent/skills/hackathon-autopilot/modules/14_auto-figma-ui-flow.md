# Auto Figma UI Flow Mode

Goal: generate a Figma-ready UI flow spec that supports the demo story.

Output must include:

- Primary user persona(s)
- User journey (happy path)
- Screen list (6–10 max for hackathon)
- For each screen:
  - goal
  - key components
  - microcopy (critical labels)
  - data shown (what makes invisible visible)
- Component checklist (design system-ish):
  - buttons, cards, alerts, charts, timeline, map, etc.

Also output a Figma/FigJam prompt:

- "Create a clean, modern hackathon demo UI..."
- includes screens as frames, with consistent layout and spacing
- includes a "Demo Mode" toggle indicator
- includes an "Explainability/Audit" panel for trustworthy AI vibes
